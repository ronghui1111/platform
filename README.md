# platform
本系统为本人首次尝试搭建spring-cloud微服务群的初步成果，采用目前市场上较为问的的组件及组件版本，微服务体系采用的spring-cloud-2021.0.1	，springbot-2.6.3为基础；
网关选择使用spring-cloud-gateway(性能，配置化网关)
用户鉴权使用sa-token（轻量，定制化改造）
服务交互使用openfeign（没什么好说的）
注册与配置中心使用（nacos)
等等等等
